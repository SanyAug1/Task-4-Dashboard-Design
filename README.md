# Task 4 - Dashboard Design

## Objective
Design an interactive dashboard for business stakeholders using Power BI.

## Dataset
Financial Sample Dataset

## Tool Used
Power BI Desktop

## Dashboard Features
- KPI Cards (Sales, Profit, Units Sold)
- Sales by Country Analysis
- Monthly Sales Trend
- Profit by Segment
- Interactive Country Slicer

## Files Included
- Power BI Dashboard (.pbix)
- Dashboard Screenshots
- PPT Summary
- Dataset

## Key Insights
- Total Sales reached 118.73M.
- Total Profit reached 16.89M.
- Government segment generated the highest profit.
- Interactive filtering allows country-wise analysis.

## Outcome
Created an interactive dashboard that helps analyze sales performance, profitability and business trends.
